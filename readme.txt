A mobile first, responsive online chess clock - New and Improved!


 ## Created using Javascript ES6, Webpack, Babel, HTML5, and CSS3. ##


 Website: [http://chessclock.netlify.com](http://chessclock.netlify.com)

 Features:
  * Mobile First
  * Responsive Design
  * Javascript ES6
  * Webpack 4
  * Babel
  * HTML5
  * CSS3
  * Sound Effects
  * CSS Sprites
